# Customer-Sales-Data-Analysis
This project analyzes sales data using Excel, SQL, and Power BI to evaluate sales, profit, customer behavior, and regional performance. Excel was used for data cleaning, SQL for data analysis, and Power BI for creating interactive dashboards. The project identifies key trends and supports data-driven business decision-making.


Project Overview

This project focuses on analyzing sales data to identify trends, evaluate profitability, and understand regional and category-wise performance. The goal is to generate meaningful business insights that support data-driven decision-making.
